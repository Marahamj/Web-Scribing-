# Web-Scribing-
It focuses on extracting comprehensive data about teams and players participating in the FIFA Club World Cup using web scraping techniques. The goal was to collect and structure over 500 rows of real-world data from the official ESPN website.
(Technical Skills)


Language: Python.

Libraries:

BeautifulSoup4: For parsing HTML and navigating the document tree.

Requests: For handling HTTP requests to the target URL.

Environment: Jupyter Notebook (.ipynb)
